i have started using git

to initiallise git in directory
    git init

to add file to tracking
    git add . 

    the . is for everything --- can be replaced by file name or folder 

to commit changes
    git commit -m "commit message"

to check for if you need to commit something ie you staged but didnt commit
    git status

to save work to github
    git push -u origin main

to clone a remote repo from the internet 
    git clone <repo_link>
